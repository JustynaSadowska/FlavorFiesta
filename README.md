# FlavorFiesta

## Project description
The application helps with everyday meal preparation, making it easier to organize the kitchen and manage recipes. It allows users to create their own recipes, browse existing ones, and quickly plan cooking tasks.

## Technologies used
- **Backend:** ASP.NET Core 9.0, Entity Framework 
- **Frontend:** React, TypeScript, Material UI, TanStack Query, Zod  
- **Database:** PostgreSQL  
- **External services:** Cloudinary, MimeKit
